CSS: Cascading Style Sheets

  • Presentational language developed to control the presentation of structured documents such as HTML file.

    • HTML has evolved into a primarily structural language, this is where CSS comes in. CSS gives an extended control over a HTML documents presentation.

    CSS styles are made up of two parts.
      Selector
    • Tells the browser which elements to style
      Declaration
    • Gives the formatting instructions

    • Color
