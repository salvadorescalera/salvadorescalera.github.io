Web Server Basics

Web sites are “Hosted” on web servers. Web servers are nothing more than computers with specialized software and hardware. All Web servers are assigned ip addresses. Web sites are stored on web servers. Web stacks are groups of software that work in conjunction with each other to build and process web sites.
