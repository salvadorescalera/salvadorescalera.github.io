Exporting design assets

File size should be as small as possible. This increases page load times and gives a better user experience.

.svg

  • Svg files are a vector image which can be made larger or smaller while retaining quality.
  • The more points the vector has the larger the file will become, try to keep images as simple as possible.

    • Bitmap

  • .jpg and .png images are bitmap images. It is important to export these images at the size you want it to appear at. Upsizing or downsizing these images will cause the image to loose its quality.
