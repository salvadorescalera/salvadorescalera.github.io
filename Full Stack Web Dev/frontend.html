Front-end Technology

Front-end design

  • The visual layer of websites and applications
  • Typically the UI, layout, typography, images and other visual elements and their styling
  • Can often refer to usability and user experience design well

    • The most common languages for creating front-end content are HTML, CSS, and JavaScript

    HTML, the language of the web

  • HTML: Hypertext Markup Language, The standard language used for structuring content on the web. HTML5 has now been adopted by the W3C and given a working charter. WHATWG and W3C continue the development.

    • Structuring HTML

  • Learning HTML is simply learning the syntax of the language, which elements are supported and the structure. Can be learned in about a day or two. Tags can be structured in various ways, they are used around the content that need editing.
